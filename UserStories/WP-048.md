# WP-048: View operation response structure with collapsible details

- **ID**: WP-048
- **Description**: As a technical user, I want a collapsible response structure preview in the operation responses list, so I can inspect response schema details without opening edit mode every time.
- **Acceptance criteria**:
  - Each response row supports expand/collapse behavior for structure preview.
  - By default, response rows are collapsed to keep the list compact.
  - Expanding a row shows the response model structure for the selected status code and media type.
  - Inline schemas are rendered as a readable field tree in the expanded area.
  - Schema references show the referenced schema structure in the expanded area when available.
  - If the schema reference is missing or unsupported, the expanded area shows a clear non-blocking message.
  - Expanding or collapsing one row does not unintentionally change other rows.
