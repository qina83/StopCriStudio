# WP-036: Delete a schema

- **ID**: WP-036
- **Description**: As a technical user, I want to delete a schema from the Schemas panel, so I can remove data models that are no longer needed in my specification.
- **Acceptance criteria**:
  - Each schema in the Schemas panel list has a "Delete" action.
  - Clicking "Delete" shows a confirmation prompt before removing the schema.
  - Confirming the deletion removes the schema from `components/schemas` immediately.
  - If `components/schemas` becomes empty after deletion, the section is removed from the specification content (no empty object left behind).
  - The Schemas panel list updates immediately to reflect the deletion.
  - Cancelling the confirmation leaves the schema intact.
