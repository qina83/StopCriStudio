# WP-049: Open schema form from response schema reference

- **ID**: WP-049
- **Description**: As a technical user, I want schema references shown in operation responses to be clickable, so I can jump directly to the referenced schema form for review or editing.
- **Acceptance criteria**:
  - A response row using a schema reference displays the referenced schema name as an interactive link element.
  - Clicking the schema reference navigates to the Schemas section.
  - After navigation, the referenced schema form opens directly in edit mode.
  - If the referenced schema does not exist, the app does not crash and shows a clear message.
  - The navigation preserves data integrity (no unsaved response data is mutated by navigation alone).
  - Keyboard activation (Enter/Space) triggers the same behavior as mouse click for accessibility.
