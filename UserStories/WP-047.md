# WP-047: Delete only the selected operation response media type

- **ID**: WP-047
- **Description**: As a technical user, I want deleting a response from an operation to remove only the selected media type entry, so I do not accidentally lose other responses defined for the same operation.
- **Acceptance criteria**:
  - When I click Delete on a specific response row (status code + media type), only that exact media type entry is removed from that status code.
  - If the deleted media type is the last one under that status code, the status code response node is removed.
  - Deleting one response row does not remove other status codes under the same operation.
  - Deleting one operation response does not affect responses of other operations or other paths.
  - The delete confirmation clearly shows the target status code and media type before confirming.
  - After deletion, the responses list updates immediately and reflects the remaining entries correctly.
