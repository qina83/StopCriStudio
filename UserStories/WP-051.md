# WP-051: Click schema item in sidebar to open schema form

- **ID**: WP-051
- **Description**: As a technical user, I want schema items in the sidebar to be clickable like path list items, so I can open and edit a specific schema directly from navigation.
- **Acceptance criteria**:
  - Each schema item in the sidebar is interactive and can be selected by click.
  - Clicking a schema item navigates to the Schemas section.
  - Clicking a schema item opens that exact schema form in edit mode.
  - The selected schema item gets visible selected-state feedback in the sidebar, consistent with path item selection behavior.
  - If the selected schema is filtered out by active search text, selection state remains consistent when the filter is cleared.
  - If a schema is deleted while selected, the sidebar selection resets safely without errors.
