# WP-044: Edit and remove operation responses

- **ID**: WP-044
- **Description**: As a technical user, I want to edit and delete existing responses on an operation, so I can keep response contracts accurate as API behavior evolves.
- **Acceptance criteria**:
  - Each response row provides edit and delete actions.
  - Editing allows updating status code, media type, description, and model source (inline or `$ref`).
  - Changing status code or media type checks uniqueness against existing combinations in the same operation.
  - Save is blocked with a clear message if the edited status code and media type combination would duplicate an existing one.
  - Deleting a media type removes only that media type entry for the status code.
  - If the last media type under a status code is deleted, the entire status code response node is removed.
  - Deleting a response does not delete reusable schemas from `components.schemas`.
  - The panel list refreshes immediately after edit or delete.
