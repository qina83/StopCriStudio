# WP-020: Export query parameters to YAML

- **ID**: WP-020
- **Description**: As a technical user, I want query parameters exported in the YAML specification, so the exported OpenAPI document includes complete query parameter documentation.
- **Acceptance criteria**:
  - Query parameters are included in exported YAML for the correct operation.
  - Exported parameters are formatted under the operation's parameters array using valid OpenAPI structure.
  - Scalar parameters export with the correct schema type.
  - Object parameters export with object schemas and nested properties.
  - Array parameters export with array schemas and items definitions.
  - Constraints such as minimum, maximum, pattern, required, default, and description are preserved in the export where applicable.
  - Nested combinations, including arrays of objects and objects containing arrays, export correctly.
  - Generated YAML is valid for the supported OpenAPI version.
