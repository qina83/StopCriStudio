# WP-008: User can load and delete saved specifications

- **ID**: WP-008
- **Description**: As a returning user, I want to load a previously saved specification from a modal list and delete specifications I no longer need, so I can manage my API documentation projects efficiently.

**Acceptance criteria**:

- Load button on Welcome page opens a modal displaying all saved specifications
- Each saved specification row shows the name, spec version, and last updated date
- Each row includes a Delete button
- Clicking Delete shows a confirmation modal asking for confirmation
- User can select a specification by clicking on the row (highlights the selection)
- Selected row is visually highlighted with a blue border and background
- An "Ok" button loads the selected specification into the editor
- A "Cancel" button closes the modal without loading anything
- After deleting a specification, the list is updated automatically
- Empty state message appears when no saved specifications exist
- Modal is centered on screen and responsive on mobile devices
- Delete confirmation modal prevents accidental deletions
